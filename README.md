# Algoritmi-e-strutture-dati-2024-25-Laboratori-
PoliTO course Algoritmi e strutture dati laboratories solutions

## What are these files for?
The files shared are the solutions to the laboratories of the **Algorithms and Data Structures** course of [Polytechnic of Turin](https://www.polito.it/), taught in the first semester of the 2^(nd) year of Bachelor Degree.


## What do these files contain?
The laboratories exercises cover all the topics taught during the course, which space from basic dynamic data structures and basic algorithms to more complicated data structures such as *graphs, hash maps, trees, queues, lists, ...* and both algorithms relative to these structures, belonging to diverse programming paradigms (*greedy, dynamic programming, divide et impera*), and algorithms which explore solutions spaces of a problem using *combinatorial calculus*, used on primitive type arrays and arrays of *structs* defined in relation to the requests of the exercise; multi-file programming is also covered in the course and it is used in some of the more advanced laboratories.
